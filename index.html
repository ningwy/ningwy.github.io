理解 Window 和 WindowManager

发表于 2017-04-04 | 分类于 Android | | 阅读次数

Window 和 WindowManager 在平常开发中不常用，但是还是会有用到的地方。Window 表示一个窗口的概念，它是一个抽象类，具体在 WindowManagerService 中由 PhoneWindow 来实现。而 WindowManager 则是开发者访问 Window 的入口，WindowManager 实现了 ViewManager，ViewManager 提供了一系列创建、移除和更新 Window 的方法：

  • addView（View view, ViewGroup.LayoutParams params）

  • removeView(View view)

  • updateViewLayout(View view, ViewGroup.LayoutParams params)

以上三个都很好懂，和一般的 View 添加删除没有什么操作上的区别。ViewGroup.LayoutParams 一般用的是 WindowManager.LayoutParams，WindowManager.LayoutParams 和平常常用的 LayoutParams 也没有多大的区别，但是其有两个比较特殊的参数，FLAGS 和 TYPE，如下：
